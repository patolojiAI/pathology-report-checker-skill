# Breast Specimen Macroscopy Guidelines

Source: AAPA Macroscopic Examination Guidelines (2017), CAP Breast Cancer Protocols
Based on: AJCC/UICC TNM 7th/8th Edition, CAP Protocol 3.3.0.0

---

## Scope

Applies to all invasive carcinomas of the breast, including DCIS with microinvasion.

---

## Procedures Covered

| Type | Description |
|------|-------------|
| **Core needle biopsy** | Multiple tissue cores (molecular considerations) |
| **Incisional biopsy** | Diagnostic tissue sample |
| **Excisional biopsy** | Lumpectomy without orientation |
| **Lumpectomy / Partial mastectomy** | Oriented breast tissue with margins |
| **Quadrantectomy** | Quadrant resection |
| **Re-excision (Complete/Global)** | All margins re-excised |
| **Re-excision (Select/Targeted)** | Specific margin(s) re-excised |
| **Simple mastectomy** | Without axillary lymph nodes |
| **Skin-sparing mastectomy** | Removal of nipple with narrow skin rim |
| **Nipple-sparing mastectomy** | No attached skin or nipple removed |
| **Modified radical mastectomy** | With axillary dissection |
| **Radical mastectomy** | With axillary dissection and pectoralis muscles |
| **Sentinel lymph node** | 1-3 nodes with tracer |
| **Axillary dissection** | Levels I, II, III lymph nodes |

## Critical Pre-Analytic Requirements

### Fixation Times (CAP Checklist Requirements)

| Parameter | Requirement | Notes |
|-----------|-------------|-------|
| **Cold ischemic time** | ≤1 hour | From tissue removal to fixation |
| **ER/PR fixation** | 6-72 hours | In 10% neutral buffered formalin |
| **HER2 fixation** | 6-72 hours | In 10% neutral buffered formalin |

### Delayed Specimen Handling

If delivery delayed or cannot process within 1 hour:
1. Bisect the tumor
2. Immerse in fixative immediately
3. Retain margin identity OR separately submit margins

### Research Tissue

For non-histologic assays (RT-PCR, etc.):
- Take in a way that allows evaluation for invasion
- Match with adjacent tissue slice for microscopic examination

- [ ] Weight (grams)
- [ ] Laterality (Right/Left)
- [ ] Three dimensions with orientation (AP / ML / SI)
- [ ] Attached skin: two dimensions + location

#### Excision / Lumpectomy
- [ ] Weight (grams)
- [ ] Laterality
- [ ] Three dimensions with orientation

#### Complete Re-excision (Global)
- [ ] Weight
- [ ] Laterality
- [ ] Three dimensions with orientation (if oriented)
- [ ] If unoriented: two surface dimensions + thickness

#### Select Re-excision (Targeted)
- [ ] Weight
- [ ] Laterality
- [ ] Two surface dimensions + thickness (as range)

#### Mastectomy
- [ ] Weight
- [ ] Laterality
- [ ] Three dimensions with orientation
- [ ] Attached skin: two dimensions
- [ ] Nipple/areola status (if present)
- [ ] Pectoralis muscle/fascia: present/absent, involvement
- [ ] Axillary tail: present/absent, dimensions
- [ ] Additional structures/abnormalities

## Specimen Integrity Assessment

### Defects and Disruptions

**Required documentation:**
1. Location of defects/disruptions
2. Relationship to tumor
3. Impact on margin assessment

**Management:**
- Apply differential ink to margins in affected areas
- Photograph defects for documentation
- Consult surgeon for clarification when needed

### Specimen Received Sectioned/Fragmented
- Document if specimen received sectioned or fragmented
- Note impact on margin evaluation

## Tumor Documentation

### Tumor Size
- [ ] Three dimensions (cm)
- Greatest dimension used for T staging

### Tumor Location
- [ ] Quadrant or clock position
- [ ] Depth from skin/deep margin

### Tumor Appearance
- [ ] Color
- [ ] Consistency (firm, soft, mucoid)
- [ ] Borders (circumscribed vs infiltrative)

### Tumor Focality

| Term | Definition |
|------|------------|
| **Multifocal** | ≥2 foci of cancer within the SAME quadrant |
| **Multicentric** | ≥2 foci of cancer in DIFFERENT quadrants (typically >5 cm apart) |

**Required documentation for multiple foci:**
- [ ] Quadrant location of each lesion
- [ ] Distance between lesions
- [ ] Size of each focus

## Margin Evaluation

### Standard Inking Protocol (6-color)

| Margin | Suggested Color |
|--------|-----------------|
| Superior | Blue |
| Inferior | Yellow |
| Medial | Green |
| Lateral | Orange |
| Anterior | Red |
| Posterior/Deep | Black |

*Document actual colors used in gross description*

### Inking Technique
- Apply to specimen surface
- Avoid penetration into specimen
- Use multiple colors for different margins
- Allow ink to dry before sectioning

### Orientation Methods
- Sutures or clips on specimen surface
- Multiple colored inks
- Specific cassettes for each margin
- Separately excised margins by surgeon

### Margin Sampling
- **Perpendicular sections**: Preferred for accurate distance measurement
- **En face sections**: Alternative, less accurate for distance
- Submit all margins closest to tumor

### Margin Status Definitions

**Positive margin:** Ink on carcinoma

**Negative margin:** No ink on carcinoma; report distance in mm

### Margin Extent Classification

| Category | Definition |
|----------|------------|
| **Focal** | One focal area of carcinoma at margin, <4 mm |
| **Minimal/Moderate** | Two or more foci of carcinoma at margin |
| **Extensive** | Carcinoma at margin over broad front (>5 mm) |

### Special Margins

#### Deep (Posterior) Margin
- Report skeletal muscle invasion (pectoralis)
- Invasive carcinoma at deep margin with muscle invasion → indication for postmastectomy radiation

#### Superficial (Anterior) Margin
- Immediately below skin
- Breast tissue may remain in skin flaps
- Document flap thickness if relevant

### Re-excision Guidance

| Margin Status | Recommendation |
|---------------|----------------|
| Close (<1 mm) or focally involved | Re-excision not mandatory; case-by-case |
| Ink positive | Re-excision required OR document reasoning |

## Sectioning Protocol

### Standard Approach
- Section perpendicular to longest axis at 3-5 mm intervals
- Document plane of sectioning
- Photograph cut surface if tumor visible

### Sampling Requirements

| Component | Minimum Sections |
|-----------|------------------|
| Tumor ≤2 cm | Submit entirely |
| Tumor >2 cm | At least 1 section per cm (minimum 3) |
| Each margin | At least 1 perpendicular section |
| Closest margin | Additional sections to demonstrate distance |
| Nipple | 1 section (sagittal) |
| Skin lesions | 1 section each |
| Lymph nodes | Entirely submitted |
| Background breast | 1-2 sections |
| Biopsy site | If identifiable |

## Radiographic Considerations

### Specimen Radiography
- Important to assess adequacy of excision
- Minimize compression (can compromise margin assessment)
- Compression devices: reserve for nonpalpable lesions (microcalcifications)

### Calcifications
- Confirm presence in specimen by radiography
- Document relationship to invasive carcinoma and DCIS
- Cancer typically at or near calcification site

### Radiographically Identifiable Markers
- Also called: "clips", "markers"
- Be aware of possible marker migration
- Correlate with patient imaging

### Techniques to Reduce Positive Margins
1. Radioactive seed localization vs. wire
2. Cavity shave excisions
3. Hematoma US-guided excisions
4. Intraoperative US by surgeon
5. Oncoplastic lumpectomy/reduction mammoplasty
6. Intraoperative margin assessment (frozen section/touch prep)

## Lymph Node Evaluation

### Regional Lymph Node Anatomy

#### Axillary Nodes (Ipsilateral)

| Level | Location |
|-------|----------|
| **Level I (low-axilla)** | Lateral to lateral border of pectoralis minor |
| **Level II (mid-axilla)** | Between medial and lateral borders of pectoralis minor; includes interpectoral (Rotter's) nodes |
| **Level III (apical axilla)** | Medial to medial margin of pectoralis minor, inferior to clavicle; also called infraclavicular nodes |

#### Other Regional Nodes
- **Internal mammary**: In intercostal spaces along sternum edge
- **Supraclavicular**: In supraclavicular fossa triangle
- **Intramammary**: Within breast tissue; classified as axillary for staging

### Sentinel Lymph Node Processing

**Definition:** First involved lymph node, identified by radiotracer or dye uptake

**Processing:**
1. Serial section at 2 mm intervals
2. If no macroscopic tumor: submit entirely
3. Document if nonsentinel node is positive when sentinel is negative

### Axillary Dissection

**Minimum nodes:** 10 lymph nodes recommended (technique-dependent)

### Lymph Node Documentation Requirements

- [ ] Total number of nodes examined (sentinel + nonsentinel)
- [ ] Number of sentinel nodes examined
- [ ] Number of nodes with macrometastases
- [ ] Greatest dimension of largest metastatic focus
- [ ] Extranodal tumor extension (present/absent)
- [ ] Site of lymph nodes
- [ ] Node dimensions (3D or range)

### Macroscopically Positive Nodes

**Documentation:**
- Cut surface description
- Macroscopic appearance of metastasis
- Size and location within node (subcapsular/intramedullary)
- Precise size of node and metastatic deposit

**Sampling:**
- Representative sampling adequate
- Sample areas of possible extranodal extension
- Submit with long axis demonstrated
- Each node evaluable and countable separately

### Macroscopically Negative Nodes

**Processing:**
- Small nodes (2-3 mm): Submit in toto
- Larger nodes: Section at 2 mm intervals, entirely submit
- Submit entire capsule (for extranodal extension evaluation)
- Large negative nodes: Submit sequentially for size calculation

### pN Classification Notes

| Category | Definition |
|----------|------------|
| **ITC** | Isolated tumor cells ≤0.2 mm OR <200 cells |
| **Micrometastasis** | >0.2 mm but ≤2.0 mm |
| **Macrometastasis** | >2.0 mm |

- ITCs excluded from positive node count but included in total evaluated
- Tumor nodules with smooth contour in regional area → classify as positive node
- Size of metastasis (not node) used for N category

## Gross vs Diagnosis Cross-Validation

### Tumor Size

| Issue | Flag | Severity |
|-------|------|----------|
| Gross size vs microscopic size differs >5mm | Size discrepancy | Warning |
| Gross tumor not identified but invasive carcinoma in diagnosis | Occult tumor | Info |
| Multiple tumors gross, single in diagnosis | Satellite vs separate | Warning |

### Margins

| Issue | Flag | Severity |
|-------|------|----------|
| Gross "close" (<5mm) but micro "positive" | Margin understaged | Warning |
| Gross margin not measured, micro positive | Missing gross data | Error |
| Ink not on margin microscopically | Inking issue | Warning |
| Different margins positive gross vs micro | Margin mismatch | Warning |

### Lymph Nodes

| Issue | Flag | Severity |
|-------|------|----------|
| Gross node count ≠ micro count | Count discrepancy | Warning |
| Grossly replaced nodes not confirmed | False positive gross | Info |
| No nodes in axillary dissection | Inadequate dissection | Error |

### Skin/Nipple

| Issue | Flag | Severity |
|-------|------|----------|
| Gross skin changes not in diagnosis | Documentation gap | Warning |
| Paget disease gross, not confirmed micro | False positive | Warning |
| Skin ulceration noted, T4 not assigned | Staging error | Error |

## QA Checklist

### Pre-Analytic
- [ ] Cold ischemic time documented (≤1 hour)
- [ ] Fixation time documented (6-72 hours)
- [ ] Fixative type documented (formalin)
- [ ] Specimen integrity assessed
- [ ] Defects documented and inked

### Macroscopic
- [ ] Weight documented
- [ ] Three dimensions with orientation
- [ ] Skin/nipple status (if present)
- [ ] All margins inked (6-color)
- [ ] Tumor size (3D)
- [ ] Tumor location (quadrant, depth)
- [ ] Distance to all margins
- [ ] Closest margin identified
- [ ] Multifocality documented
- [ ] Radiographic correlation (if applicable)

### Lymph Nodes
- [ ] Total count
- [ ] Sentinel vs. non-sentinel distinguished
- [ ] Metastasis size documented
- [ ] Extranodal extension assessed
- [ ] ITC/micro/macro classification
