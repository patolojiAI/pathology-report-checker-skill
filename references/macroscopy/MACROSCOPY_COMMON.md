# Macroscopy/Gross Description Guidelines

Based on AAPA Macroscopic Examination Guidelines, CAP Cancer Protocols, and AJCC Staging Manual.

## Purpose

This reference ensures:
1. Gross descriptions contain all required elements for staging
2. Gross findings are consistent with microscopic diagnosis
3. Discrepancies between macroscopy and diagnosis are flagged

---

## Universal Gross Description Elements

Every cancer resection specimen gross description should include:

### Core Elements (Required)

| Element | Description | Why Important |
|---------|-------------|---------------|
| **Specimen identification** | Type of specimen, laterality, anatomic site | Correct specimen orientation |
| **Specimen dimensions** | Overall size in 3 dimensions (cm) | Documentation completeness |
| **Tumor location** | Anatomic location within specimen | Staging (some sites) |
| **Tumor size** | Maximum dimension in cm | T staging |
| **Tumor appearance** | Color, consistency, borders | Tumor characterization |
| **Distance to margins** | Closest margin(s) with measurement | Resection adequacy |
| **Margin identification** | Margins inked with colors specified | Microscopic correlation |
| **Lymph nodes** | Number found, sizes, appearance | N staging |
| **Additional lesions** | Separate tumors, polyps, other findings | Complete documentation |
| **Tissue submitted** | Sections taken, cassette designations | Sampling adequacy |

### Conditional Elements

| Element | When Required |
|---------|---------------|
| **Skin involvement** | Breast, melanoma specimens |
| **Serosal appearance** | GI resections, ovarian tumors |
| **Vessel involvement** | Visible tumor in vessels |
| **Perforation** | GI specimens |
| **Mesorectal integrity** | TME specimens |
| **Capsule status** | Encapsulated organs (kidney, adrenal, thyroid) |

## Gross vs Diagnosis Cross-Validation

### Tumor Size Discrepancies

| Issue | Detection Rule | Severity |
|-------|----------------|----------|
| Gross size vs microscopic size differs >20% | Compare measurements | Warning |
| Gross size missing but required for pT | pT staging without gross size | Error |
| Multiple sizes reported inconsistently | Different sizes in gross vs diagnosis | Warning |

### Margin Discrepancies

| Issue | Detection Rule | Severity |
|-------|----------------|----------|
| Gross "close" but micro "positive" | Margin status mismatch | Warning |
| Gross distance vs micro distance differs significantly | >2mm discrepancy | Warning |
| Margin not mentioned in gross but positive in diagnosis | Missing gross documentation | Error |
| Ink color not specified but margin positive | Missing orientation | Warning |

### Lymph Node Discrepancies

| Issue | Detection Rule | Severity |
|-------|----------------|----------|
| Gross count vs micro count differs | Number mismatch | Warning |
| Gross "no nodes found" but diagnosis reports nodes | Major discrepancy | Error |
| Inadequate node count for staging | Below minimum threshold | Warning |
| Grossly positive nodes not confirmed micro | Reactive vs metastatic | Info |

### Tumor Location/Extent Discrepancies

| Issue | Detection Rule | Severity |
|-------|----------------|----------|
| Gross "confined to organ" but micro shows extension | Understaging risk | Error |
| Gross serosal involvement not mentioned in diagnosis | Documentation gap | Warning |
| Perforation noted gross but not in diagnosis | Critical finding | Error |
| Gross vessel involvement not confirmed micro | Staging impact | Warning |

### Specimen Integrity Issues

| Issue | Detection Rule | Severity |
|-------|----------------|----------|
| Gross notes specimen fragmented/disrupted | May affect margin assessment | Warning |
| Orientation not specified | Cannot assess margins | Error |
| Inking not described | Cannot verify margins | Error |

## Quality Flags

### Critical (Must Address)

- Tumor size not documented
- Margins not described
- No lymph nodes found in radical resection
- Orientation not specified for excision specimens
- Gross vs diagnosis tumor location mismatch

### Major (Should Address)

- Lymph node count below minimum
- Margin distances not measured
- Serosal status not documented
- Gross description incomplete for specimen type
- Size discrepancy >5mm between gross and micro

### Minor (Consider)

- Tumor configuration not described
- Specimen weight not recorded
- Cassette key incomplete
- Additional findings not characterized

## Cross-Validation Examples

### Example 1: Size Mismatch
**Gross:** "Tumor measures 2.5 x 2.0 x 1.8 cm"
**Diagnosis:** "Invasive ductal carcinoma, 3.2 cm"
**Flag:** ⚠️ Size discrepancy 0.7cm (28%) - verify measurement

### Example 2: Margin Inconsistency
**Gross:** "Tumor is 0.3 cm from the posterior margin"
**Diagnosis:** "Posterior margin: Positive for carcinoma"
**Flag:** ❌ Gross distance suggests clear margin, but micro shows positive

### Example 3: Node Count Mismatch
**Gross:** "15 lymph nodes identified"
**Diagnosis:** "Lymph nodes: 0/12 with metastatic carcinoma"
**Flag:** ⚠️ Node count discrepancy (15 gross vs 12 micro)

### Example 4: Missing Critical Element
**Gross:** [No mention of mesorectal quality]
**Specimen:** Total mesorectal excision
**Flag:** ❌ Mesorectal grading required for TME specimens

### Example 5: Extension Discrepancy
**Gross:** "Tumor appears confined to pancreatic parenchyma"
**Diagnosis:** "Carcinoma extends into peripancreatic soft tissue (pT3)"
**Flag:** ⚠️ Gross underestimated tumor extent

## Implementation Notes

1. **Extract gross findings first** before analyzing diagnosis
2. **Identify specimen type** to determine required elements
3. **Match elements** between gross and diagnosis sections
4. **Flag discrepancies** with appropriate severity
5. **Consider measurement variability** (±2mm acceptable for margins)
6. **Document missing elements** that should be present for specimen type
