# Gastric Carcinoma - Diagnosis Guidelines

Based on CAP Cancer Protocol for Stomach and ICCR Gastric Cancer Dataset.

## Application

This protocol applies to:
- All carcinomas arising in the stomach
- EGJ carcinomas with tumor midpoint >2cm into proximal stomach
- Cardia carcinomas without EGJ involvement

**Excluded** (use separate protocols):
- EGJ carcinomas with midpoint ≤2cm into stomach → use Esophagus protocol
- Well-differentiated neuroendocrine tumors → use Stomach NET protocol
- Lymphomas, GISTs, sarcomas

## Histologic Types (WHO 5th Edition)

### Common Types

| Type | Description |
|------|-------------|
| **Tubular adenocarcinoma** | Dilated/slit-like tubules, most common |
| **Papillary adenocarcinoma** | Papillary structures with fibrovascular cores |
| **Mucinous adenocarcinoma** | >50% extracellular mucin pools |
| **Poorly cohesive carcinoma** | Including signet ring cell type |
| **Mixed adenocarcinoma** | Combination of above types |

### Rare Types

| Type | Description |
|------|-------------|
| Adenosquamous carcinoma | ≥25% squamous component |
| Squamous cell carcinoma | Pure squamous |
| Undifferentiated carcinoma | No glandular/squamous differentiation |
| Hepatoid adenocarcinoma | AFP-producing, hepatocyte-like |
| Carcinoma with lymphoid stroma | Medullary/lymphoepithelioma-like |
| Micropapillary adenocarcinoma | Micropapillary pattern |

---

## Histologic Grade

| Grade | Description | Differentiation |
|-------|-------------|-----------------|
| **G1** | Well differentiated | >95% gland formation |
| **G2** | Moderately differentiated | 50-95% gland formation |
| **G3** | Poorly differentiated | <50% gland formation |
| **G4** | Undifferentiated | No glandular differentiation |

**Note:** Poorly cohesive carcinomas (including signet ring cell) are classified as G3.

## Laurén Classification

| Type | Features | Prognosis |
|------|----------|-----------|
| **Intestinal** | Gland-forming, cohesive, expanding growth | Better |
| **Diffuse** | Non-cohesive, infiltrative, signet ring cells | Worse |
| **Mixed** | Both patterns present | Intermediate |
| **Indeterminate** | Cannot be classified | Variable |

## pT Classification (AJCC 8th Edition)

| Category | Definition |
|----------|------------|
| **pTX** | Primary tumor cannot be assessed |
| **pT0** | No evidence of primary tumor |
| **pTis** | Carcinoma in situ / high-grade dysplasia |
| **pT1a** | Tumor invades lamina propria or muscularis mucosae |
| **pT1b** | Tumor invades submucosa |
| **pT2** | Tumor invades muscularis propria |
| **pT3** | Tumor penetrates subserosal connective tissue without invasion of visceral peritoneum or adjacent structures |
| **pT4a** | Tumor invades serosa (visceral peritoneum) |
| **pT4b** | Tumor invades adjacent structures |

### Adjacent Structures (pT4b)

- Spleen
- Transverse colon
- Liver
- Diaphragm
- Pancreas
- Abdominal wall
- Adrenal gland
- Kidney
- Small intestine
- Retroperitoneum

**Note:** Intramural extension to duodenum or esophagus is classified by depth of greatest invasion in any of these sites.

---

## pN Classification (AJCC 8th Edition)

| Category | Definition |
|----------|------------|
| **pNX** | Regional lymph nodes cannot be assessed |
| **pN0** | No regional lymph node metastasis |
| **pN1** | Metastasis in 1-2 regional lymph nodes |
| **pN2** | Metastasis in 3-6 regional lymph nodes |
| **pN3a** | Metastasis in 7-15 regional lymph nodes |
| **pN3b** | Metastasis in 16 or more regional lymph nodes |

### Lymph Node Requirements

- **Minimum nodes for staging:** 16 lymph nodes
- ITC (isolated tumor cells ≤0.2mm) = pN0
- Tumor deposits without residual node = count as positive nodes

### Regional Lymph Node Stations

| Location | Stations |
|----------|----------|
| **Greater curvature** | Greater omental, gastroduodenal, gastroepiploic, pyloric |
| **Lesser curvature** | Lesser omental, left gastric, cardioesophageal, common hepatic, celiac, hepatoduodenal |
| **Splenic** | Pancreaticosplenic, splenic hilar |

## Margin Assessment

### Margin Types

| Margin | Description |
|--------|-------------|
| **Proximal** | Esophageal/cardia margin |
| **Distal** | Duodenal/pyloric margin |
| **Radial** | Non-peritonealized soft tissue (lesser/greater omentum) |

### Margin Status

| Status | Definition |
|--------|------------|
| **Negative** | No tumor at inked margin |
| **Positive** | Tumor at inked margin |
| **Close** | Tumor within 1-2mm of margin |

### R Classification

| Category | Definition |
|----------|------------|
| **R0** | No residual tumor |
| **R1** | Microscopic residual tumor |
| **R2** | Macroscopic residual tumor |

## Treatment Effect (Post-Neoadjuvant)

### Tumor Regression Grade

| Grade | Description |
|-------|-------------|
| **0 (Complete)** | No viable tumor cells |
| **1 (Near complete)** | Single cells or rare small groups |
| **2 (Partial)** | Residual cancer with evident regression |
| **3 (Poor/None)** | Minimal or no regression |

### ypTNM Staging

- Use "y" prefix for post-treatment staging
- ypT0 if no residual tumor
- Size based on largest contiguous focus of residual tumor

## Cross-Validation Rules

### pT vs Depth

| Finding | Expected pT |
|---------|-------------|
| Lamina propria only | pT1a |
| Submucosa | pT1b |
| Muscularis propria | pT2 |
| Subserosa without serosal involvement | pT3 |
| Serosal surface involved | pT4a |
| Adjacent organ invaded | pT4b |

### pN vs Node Count

| Positive Nodes | Expected pN |
|----------------|-------------|
| 0 | pN0 |
| 1-2 | pN1 |
| 3-6 | pN2 |
| 7-15 | pN3a |
| ≥16 | pN3b |

### Node Adequacy

| Finding | Flag |
|---------|------|
| <16 nodes examined | Warning: May understage pN |
| 0 nodes in gastrectomy | Error: Inadequate sampling |

## Biomarker Requirements

### Required Testing

| Marker | Indication | Method |
|--------|------------|--------|
| **HER2** | All gastric/GEJ adenocarcinomas (advanced/metastatic) | IHC, FISH if 2+ |
| **MSI/MMR** | All gastric adenocarcinomas | IHC or PCR |
| **PD-L1** | Advanced/metastatic disease | IHC (CPS score) |

### HER2 Scoring (Gastric-specific)

| Score | Surgical | Biopsy | Interpretation |
|-------|----------|--------|----------------|
| **0** | No staining or <10% membrane | No staining | Negative |
| **1+** | Faint membrane in ≥10% | Faint membrane in cluster ≥5 cells | Negative |
| **2+** | Weak-moderate complete/basolateral membrane in ≥10% | Complete/basolateral membrane in cluster ≥5 cells | Equivocal → FISH |
| **3+** | Strong complete/basolateral membrane in ≥10% | Strong membrane in cluster ≥5 cells | Positive |

**Note:** Gastric HER2 scoring differs from breast - basolateral staining counts!
