# Gastric Specimen Macroscopy Guidelines

Source: AAPA Macroscopic Examination Guidelines (2017), CAP Gastric Cancer Protocols
Based on: AJCC/UICC TNM 7th/8th Edition, CAP Protocol 3.3.0.0

---

## Scope

Applies to all carcinomas that arise in the stomach.

**Excluded:**
- Tumors of the esophagogastric junction
- Well-differentiated neuroendocrine tumors (carcinoid tumors)
- Lymphomas
- Sarcomas

---

## Procedures Covered

| Procedure | Description |
|-----------|-------------|
| **Endoscopic mucosal resection (EMR)** | Local mucosal excision |
| **Endoscopic submucosal dissection (ESD)** | En bloc mucosal/submucosal excision |
| **Partial gastrectomy, proximal** | Cardia/fundus resection |
| **Partial gastrectomy, distal** | Antrum/pylorus resection |
| **Partial gastrectomy, other** | Other partial resections |
| **Total gastrectomy** | Complete stomach removal |

---

## Terminology Clarification

### Radial vs. Omental Margins

**Radial margins** for stomach = **Omental margins** (lesser and greater)

- Lesser omental margin: hepatoduodenal and hepatogastric ligaments
- Greater omental margin: greater omentum

---

## Molecular Considerations

### HER2 Testing

**ToGA Trial Evidence:**
- Anti-HER2 therapy (trastuzumab/Herceptin) prolongs survival in HER2+ gastric/GEJ adenocarcinoma
- ~20% of gastric carcinomas show HER2 overexpression/amplification
- More common in:
  - Intestinal type tumors
  - Proximal tumors

**Treatment Eligibility:**
- Inoperable locally advanced tumors
- Recurrent tumors
- Metastatic adenocarcinoma

**HER2+ Definition:**
- IHC score 3+ (unequivocal overexpression), OR
- FISH/CISH amplification

### Fixation Requirements

| Parameter | Requirement |
|-----------|-------------|
| Fixative | Formalin (preferred) |
| Minimum fixation | 8 hours |
| Maximum fixation | 48 hours |

**Document in macroscopic description:**
- Fixative used
- Duration of fixation

## Specimen Integrity

### Assessment

Document how specimen was received:
- **Intact**
- **Previously opened**
- **Focal wall disruption** (describe location, relation to tumor and margins)
- **Diffuse disruption**

### Defect Documentation

Include:
1. Anatomic location of defects/disruptions
2. Relationship to tumor
3. Relationship to margins

**If defects hinder margin assessment:**
- Consult surgeon
- Differentially ink affected areas
- Document in report

## Tumor Documentation

### Tumor Size

- [ ] Three dimensions

### Tumor Site

| Region | Subsite |
|--------|---------|
| **Cardia** | GE junction area |
| **Fundus** | Anterior wall, Posterior wall |
| **Body** | Anterior wall, Posterior wall, Lesser curvature, Greater curvature |
| **Antrum** | Anterior wall, Posterior wall, Lesser curvature, Greater curvature |
| **Pylorus** | Pyloric canal |
| **Other** | Specify |

### Multiple Tumors

- Describe specific site of each tumor
- Document relation to margins for each

### Tumor Configuration

| Type | Description |
|------|-------------|
| **Exophytic/polypoid** | Protrudes into lumen |
| **Fungating** | Irregular raised mass |
| **Infiltrative** | Spreads through wall |
| **Ulcerating** | Central ulceration |
| **Ulcerating and infiltrating** | Combined pattern |
| **Diffusely infiltrative** | Linitis plastica pattern |

### Serosal Surface Examination

- [ ] Examine for tumor penetration
- [ ] Ink area overlying tumor
- [ ] Cross-section tumor
- [ ] Estimate depth of invasion
- [ ] Document involvement of adjacent structures
- [ ] Record other mucosal abnormalities

## pT Classification (AJCC 7th/8th Edition)

| Category | Definition |
|----------|------------|
| **pTX** | Primary tumor cannot be assessed |
| **pT0** | No evidence of primary tumor |
| **pTis** | Carcinoma in situ (including high-grade dysplasia); intraepithelial without lamina propria invasion |
| **pT1** | Tumor invades lamina propria, muscularis mucosae, or submucosa |
| **pT1a** | Invades lamina propria or muscularis mucosae |
| **pT1b** | Invades submucosa |
| **pT2** | Tumor invades muscularis propria |
| **pT3** | Penetrates subserosal connective tissue without visceral peritoneum or adjacent structure invasion |
| **pT4** | Invades serosa (visceral peritoneum) or adjacent structures |
| **pT4a** | Invades serosa (visceral peritoneum) |
| **pT4b** | Invades adjacent structures |

### T1 Substratification Note

- T1a vs. T1b based on lymph node metastasis frequency
- Important for limited procedure treatment decisions

### T2 vs. T3 Clarification

Tumor penetrating muscularis propria with extension into:
- Gastrocolic ligament → **T3** (no peritoneal perforation)
- Gastrohepatic ligament → **T3** (no peritoneal perforation)
- Greater or lesser omentum → **T3** (no peritoneal perforation)

**If peritoneum covering these structures is perforated → T4**

### T4b Adjacent Structures

- Spleen
- Transverse colon
- Liver
- Diaphragm
- Pancreas
- Abdominal wall
- Adrenal gland
- Kidney
- Small intestine
- Retroperitoneum

**Intramural extension:** Into duodenum or esophagus → classify by deepest invasion in any site

## Prognostic Factors

### Most Important Factors

1. **Treatment** (major factor)
2. **Regional lymphatic spread** (most powerful prognostic factor for resected tumors)
3. **Depth of invasion (T)**

### Additional Factors (for complete resection)

- Tumor location (proximal less favorable than distal)
- Histologic grade
- Lymphovascular invasion
- Lauren classification (intestinal vs. diffuse)

## Margins

### Endoscopic Mucosal Resection

| Margin | Documentation |
|--------|---------------|
| **Deep margin** | Distance from tumor |
| **Mucosal margins** | Distance from tumor (all directions) |

### Partial and Total Gastrectomy

| Margin | Documentation |
|--------|---------------|
| **Proximal margin** | Distance from tumor |
| **Distal margin** | Distance from tumor |
| **Omental margins** | Distance from tumor |
| - Lesser omental | (hepatoduodenal, hepatogastric ligaments) |
| - Greater omental | |

### Margin Documentation

**If all margins grossly uninvolved:**
- Specify closest margin
- Record distance from closest margin
- Demonstrate for microscopic evaluation

**If any margin grossly positive:**
- Specify location of positive margin(s)
- Demonstrate involvement for microscopic evaluation

### Radial (Circumferential) Margin

**Definition:** Nonperitonealized soft tissue margins closest to deepest tumor penetration

**In stomach:**
- Lesser omental margin (hepatoduodenal + hepatogastric ligaments)
- Greater omental margin

**Inking:**
- Mark margin(s) closest to tumor with ink
- Document ink designations in macroscopic description

## Lymph Node Evaluation

### pN Classification (AJCC 7th/8th Edition)

| Category | Definition |
|----------|------------|
| **pNX** | Cannot be assessed |
| **pN0** | No regional lymph node metastasis |
| **pN1** | 1-2 positive perigastric nodes |
| **pN2** | 3-6 positive perigastric nodes |
| **pN3** | >6 positive lymph nodes |
| **pN3a** | 7-15 positive regional nodes |
| **pN3b** | ≥16 positive regional nodes |

### Minimum Lymph Node Count

**Recommended: ≥16 regional nodes**

However, pN0 may be assigned based on actual number evaluated microscopically.

### Regional Lymph Node Areas

| Area | Node Groups |
|------|-------------|
| **Greater Curvature** | Greater curvature, greater omental, gastroduodenal, gastroepiploic, pyloric, pancreaticoduodenal |
| **Pancreatic/Splenic** | Pancreaticolienal, peripancreatic, splenic |
| **Lesser Curvature** | Lesser curvature, lesser omental, left gastric, cardioesophageal, common hepatic, celiac, hepatoduodenal |

### Non-Regional Nodes (= M1)

- Retroperitoneal
- Portal
- Retropancreatic
- Mesenteric
- Para-aortic

### Lymph Node Submission

#### Macroscopically Positive Nodes

- [ ] Count and size range (3D)
- [ ] One representative section per node
- [ ] Include extranodal extension if present
- [ ] Note anatomic location (if known)
- [ ] Submit with long axis demonstrated
- [ ] Ensure accurate count possible

#### Macroscopically Negative/Equivocal Nodes

- [ ] Count and size range (3D)
- [ ] Submit ALL nodes from complete dissection
- [ ] Note anatomic location (if known)
- [ ] Small nodes: submit in toto
- [ ] Large nodes: bivalve or serial section, submit entirely
- [ ] Submit with long axis demonstrated
- [ ] Ensure accurate count possible

## Distant Metastasis (pM)

| Category | Definition |
|----------|------------|
| **pM0** | No distant metastasis |
| **pM1** | Distant metastasis |

### Common Metastatic Sites

1. Liver (most common)
2. Peritoneal surfaces
3. Non-regional/distant lymph nodes
4. CNS (less frequent)
5. Pulmonary (less frequent)

### Classification Notes

| Finding | Classification |
|---------|----------------|
| Direct extension to liver, transverse colon, pancreas, diaphragm | **pT4** (not metastasis) |
| Positive peritoneal cytology | **M1** (metastatic disease) |
| Tumor nodules in perigastric fat without lymph node tissue | **Regional LN metastasis** |
| Nodules implanted on peritoneal surfaces | **M1** (distant metastasis) |

## QA Checklist

### Pre-Analytic
- [ ] Fixative documented (formalin)
- [ ] Fixation duration documented (8-48 hours for HER2)
- [ ] Specimen integrity assessed
- [ ] Defects documented

### Specimen Documentation
- [ ] Three dimensions
- [ ] Greater/lesser curvature lengths
- [ ] Proximal/distal margin circumferences
- [ ] Wall thickness
- [ ] Portion resected (if partial)
- [ ] Attached structures documented

### Tumor Assessment
- [ ] Three dimensions
- [ ] Site documented (region + subsite)
- [ ] Configuration documented
- [ ] Serosal surface examined and inked
- [ ] Depth of invasion estimated
- [ ] Adjacent structure involvement assessed
- [ ] Other mucosal abnormalities documented

### Margins
- [ ] All margins documented with distances
- [ ] Proximal margin
- [ ] Distal margin
- [ ] Lesser omental margin
- [ ] Greater omental margin
- [ ] Closest margin identified
- [ ] Positive margins specified (if applicable)

### Lymph Nodes
- [ ] Total count documented
- [ ] Positive count documented
- [ ] Minimum 16 nodes attempted
- [ ] Size range documented
- [ ] Extranodal extension assessed
- [ ] Regional vs. non-regional distinguished

### Molecular
- [ ] HER2 testing consideration noted
- [ ] Fixation requirements met (8-48 hours)
