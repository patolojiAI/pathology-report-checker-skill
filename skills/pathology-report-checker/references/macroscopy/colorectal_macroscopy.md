# Colorectal Specimen Macroscopy Guidelines

Source: AAPA Macroscopic Examination Guidelines (2017), CAP Colorectal Cancer Protocols
Based on: AJCC/UICC TNM 7th/8th Edition, CAP Protocol 3.3.0.0

---

## Scope

Applies to primary carcinoma arising from the colon and rectum.

**Excluded:**
- Tumors of the appendix
- Tumors of the anus
- Well-differentiated neuroendocrine tumors (carcinoids)
- Sarcomas
- Lymphomas

---

## Procedures Covered

| Procedure | Description |
|-----------|-------------|
| **Polypectomy** | Excisional biopsy |
| **Transanal disk excision** | Local excision of rectal tumor without anus/sphincter removal |
| **Transanal endoscopic microsurgery (TEM)** | Local excision technique |
| **Right hemicolectomy** | Cecum, ascending colon, hepatic flexure |
| **Transverse colectomy** | Transverse colon |
| **Left hemicolectomy** | Splenic flexure, descending colon |
| **Sigmoidectomy** | Sigmoid colon |
| **Low anterior resection (LAR)** | Upper 2/3 rectum, sphincter preserved |
| **Abdominoperineal resection (APR)** | Distal 1/3 rectum with anus |
| **Total colectomy** | Entire colon |
| **Total mesorectal excision (TME)** | Rectum with complete mesorectum |

## Molecular Considerations

### Biomarkers

| Test | Purpose |
|------|---------|
| **MSI / MMR IHC** | Lynch syndrome detection (HNPCC) |
| **KRAS mutation** | Anti-EGFR therapy eligibility (cetuximab, panitumumab) |
| **NRAS mutation** | Anti-EGFR therapy eligibility |
| **BRAF V600E** | Prognosis (poorer if positive) |

### Tissue Requirements

- Formalin-fixed, paraffin-embedded tissue
- Include tumor section with adjoining normal colon (internal control for MMR)
- Sample non-necrotic tumor areas
- Primary or metastatic tumor tissue acceptable for KRAS/NRAS/BRAF

### Documentation

- [ ] Fixative used (formalin preferred)
- [ ] Duration of fixation
- [ ] Cold ischemic time (if possible)

## Specimen Documentation

### Polypectomy

- [ ] Three dimensions
- [ ] Sessile or pedunculated
- [ ] Stalk length and diameter (if pedunculated)
- [ ] Ink applied to cauterized base
- [ ] Sectioning method (bisect or serial section)

**Sectioning:** Follow vertical plane of stalk to maximize margin evaluation

### Transanal Disk Excision

- [ ] Two dimensions and excisional depth
- [ ] Orientation (if provided)

### Resection Specimens

**Required measurements:**

| Component | Measurements |
|-----------|--------------|
| Ileum (if present) | Length and diameter/circumference |
| Appendix (if present) | Length and diameter |
| Colon | Length and diameter/circumference |
| Mesentery | Three dimensions |
| Omentum (if present) | Three dimensions |

**Additional documentation:**
- [ ] Anatomic structures identified
- [ ] Attached fibroadipose tissue (measured or minimal/moderate/extensive)
- [ ] True margin vs. peritonealized surface
- [ ] Anastomoses present
- [ ] Additional structures

## Specimen Integrity

### Polypectomy/Excision

Document as:
- **Intact**
- **Fragmented** (include number of fragments)
- **Disrupted**

### Resection

- [ ] Proximal and distal margin status (how received)
- [ ] Defects or disruptions identified and inked
- [ ] Location relative to margins and tumor
- [ ] Mesorectal completeness (for rectal specimens)

## Mesorectal Envelope Assessment

The completeness of mesorectum predicts local recurrence and distant metastasis.

### Grading Scale

| Grade | Criteria |
|-------|----------|
| **COMPLETE** | Intact bulky mesorectum with smooth surface; only minor irregularities; no defects >5 mm depth; no coning toward distal margin; smooth circumferential margin on transverse section |
| **NEARLY COMPLETE** | Moderate bulk; irregularities with defects >5 mm but not to muscularis propria; no visible muscularis propria (except levator ani insertion) |
| **INCOMPLETE** | Little bulk; defects down to muscularis propria; very irregular circumferential margin on transverse section |

**Scoring:** Entire specimen scored according to worst area

### Mandatory Reporting

- Distance from tumor to nearest circumferential margin
- Distance from tumor to closest mucosal margin
- Proximity of tumor to circumferential margin (even if in lymph nodes or tumor deposits)

## Tumor Documentation

### Tumor Size

- [ ] Three dimensions
- [ ] Multiple tumors: measure each separately
- [ ] Percentage of circumference involved

### Tumor Site

| Subsite | Notes |
|---------|-------|
| Cecum | |
| Ileocecal valve | |
| Right (ascending) colon | |
| Hepatic flexure | |
| Transverse colon | |
| Splenic flexure | |
| Left (descending) colon | |
| Sigmoid colon | |
| Rectosigmoid | State if above/below peritoneal reflection |
| Rectum | State if above/below peritoneal reflection |

### Tumor Classification Rules

**Rectum definition:**
- ≤16 cm from anal verge, OR
- Any part within colon supplied by superior rectal artery
- Distal border: anorectal ring (~2 cm from dentate line)

**Rectosigmoid:** When differentiation between rectum and sigmoid is not possible

**Border tumors:** Classified to more involved subsite; if equal involvement → "overlapping"

### Tumor Configuration

Document:
- [ ] Exophytic/polypoid
- [ ] Ulcerating
- [ ] Annular/constrictive
- [ ] Infiltrative
- [ ] Cut surface appearance
- [ ] Relationship to adjacent mucosa

### Recurrent Tumor

- Document relationship to anastomosis line
- Recurrent tumors at colon-colon anastomoses → classified as proximal anastomotic segment

### Perforation

**Tumor perforation (T4):**
- Measure perforation to margins

**Non-tumor perforation:**
- Document if proximal or distal to tumor
- Proximal perforations in obstructing tumors → peritonitis/sepsis risk
- Must document that perforation is NOT from tumor penetration

## Depth of Invasion

### pT Classification (AJCC 7th/8th Edition)

| Category | Definition |
|----------|------------|
| **pTX** | Primary tumor cannot be assessed |
| **pT0** | No evidence of primary tumor |
| **pTis** | Carcinoma in situ (intraepithelial, with or without invasion of lamina propria/muscularis mucosa) |
| **pT1** | Tumor invades submucosa |
| **pT2** | Tumor invades muscularis propria |
| **pT3** | Tumor invades through muscularis propria into pericolorectal tissues |
| **pT4a** | Tumor penetrates visceral peritoneum |
| **pT4b** | Tumor directly invades or is adherent to other organs/structures |

### pT4b Notes

- Direct invasion of other colorectal segments via serosa/mesocolon = pT4b
- Adherent structures WITHOUT malignant cells in adhesion → classify as pT1-pT4a based on wall invasion depth
- Intramural extension to adjacent subsite does NOT affect pT classification

### Sectioning for Depth

- [ ] Section at area of deepest penetration
- [ ] Include relationship to adjacent non-neoplastic mucosa
- [ ] Document extent within wall (submucosa, muscularis propria, serosa)
- [ ] Document relation to peritonealized and non-peritonealized surfaces

## Treatment Effect

Rectal cancer responds well to neoadjuvant therapy.

### Gross Findings Post-Treatment

- Tumor may not be grossly evident
- Tumor site may be fibrotic
- Mucin pools may be present (represent eradicated tumor, not residual)

### Sampling

- Thoroughly sample tumor site (may be marked by tattoo)
- Microscopic tumor can be present even when macroscopically negative

### Residual Tumor Classification

| Category | Definition |
|----------|------------|
| **R0** | Complete resection, all margins negative |
| **R1** | Microscopic margin involvement (grossly negative) |
| **R2** | Macroscopic residual tumor not resected |

## Margins

### Polypectomy Margins

- [ ] Peripheral mucosa
- [ ] Deep margin
- [ ] Stalk margin (if applicable)

**Note:** Margin status of malignant polypectomy = independent prognostic indicator

### Transanal Disk Excision

- [ ] Distance to nearest mucosal margin
- [ ] Distance to deep margin
- [ ] Specify if oriented

### Resection Margins

| Margin | Documentation |
|--------|---------------|
| **Proximal** | Distance from tumor |
| **Distal** | Distance from tumor |
| **Circumferential (radial)** | Distance from tumor; positive if ≤1 mm |

### Inking Protocol

- Ink margin(s) closest to tumor
- Examine serosal surface for puckering/tumor involvement
- Document ink designations in report

### Margin Sampling

- En face sections parallel to margin, OR
- Longitudinal sections perpendicular to margin

### Circumferential Margin Rules

| Distance | Status |
|----------|--------|
| >1 mm from inked non-peritonealized surface | **Negative** |
| ≤1 mm from inked non-peritonealized surface | **Positive** |

**Includes:**
- Direct tumor extension
- Tumor within lymph node
- Tumor deposits

**Note:** If positivity based on intranodal tumor, state this

### Adequate Margins

| Margin Type | Adequate Distance |
|-------------|-------------------|
| Distal (rectal/rectosigmoid) | ≥2 cm (≥1 cm may be sufficient for pT1-T2) |
| Proximal or distal (any) | ≥5 cm → low anastomotic recurrence |

### Total Mesorectal Excision Margins

- [ ] Proximal margin
- [ ] Distal margin
- [ ] Circumferential resection margin (CRM)

**CRM is the most critical factor for local recurrence prediction**

### Special Considerations

- Serosal (peritonealized) surface ≠ surgical margin
- Areas of possible serosal involvement → well sampled (positivity = pT4)
- Colon completely covered by peritonealized surface → only mesenteric margin is true margin

---

## Uninvolved Colon

**Always take sections of uninvolved colon**

Document and submit:
- Additional polyps/adenomas
- Signs of Crohn's disease
- Signs of ulcerative colitis

## Lymph Node Evaluation

### pN Classification (AJCC 7th/8th Edition)

| Category | Definition |
|----------|------------|
| **pNX** | Cannot be assessed |
| **pN0** | No regional lymph node metastasis |
| **pN1** | Metastasis in 1-3 regional lymph nodes |
| **pN1a** | 1 positive node |
| **pN1b** | 2-3 positive nodes |
| **pN1c** | Tumor deposits without regional node metastasis |
| **pN2a** | Metastasis in 4-6 regional lymph nodes |
| **pN2b** | Metastasis in ≥7 regional lymph nodes |

### Minimum Lymph Node Count

| Scenario | Minimum |
|----------|---------|
| Without neoadjuvant therapy | 12-14 nodes |
| With neoadjuvant therapy | Fewer acceptable |
| Palliative resection | Fewer acceptable |

**If <12 nodes identified:** Re-examine specimen with or without visual enhancement techniques

### Regional Lymph Nodes by Subsite

| Subsite | Regional Nodes |
|---------|----------------|
| **Cecum** | Pericolic, anterior cecal, posterior cecal, ileocolic, right colic |
| **Ascending colon** | Pericolic, ileocolic, right colic, middle colic |
| **Hepatic flexure** | Pericolic, middle colic, right colic |
| **Transverse colon** | Pericolic, middle colic |
| **Splenic flexure** | Pericolic, middle colic, left colic, inferior mesenteric |
| **Descending colon** | Pericolic, left colic, inferior mesenteric, sigmoid |
| **Sigmoid colon** | Pericolic, inferior mesenteric, superior rectal, sigmoidal, sigmoid mesenteric |
| **Rectosigmoid** | Pericolic, perirectal, left colic, sigmoid mesenteric, sigmoidal, inferior mesenteric, superior rectal, middle rectal |
| **Rectum** | Perirectal (mesorectal/paraproctal), sacral promontory, superior/middle/inferior rectal, sigmoid mesenteric, inferior mesenteric, lateral sacral, presacral, internal iliac |

### Macroscopically Positive Nodes

- [ ] Count and size range (3D)
- [ ] One representative section per node
- [ ] Include extranodal extension if present
- [ ] Anatomic location (if known)
- [ ] Submit with long axis demonstrated
- [ ] Ensure accurate count possible

### Macroscopically Negative/Equivocal Nodes

- [ ] Count and size range (3D)
- [ ] Submit ALL nodes from complete dissection
- [ ] Include circumferential margin relationship if close (<2 mm)
- [ ] Anatomic location (if known)
- [ ] Small nodes: submit in toto
- [ ] Larger nodes: bivalve or serial section, submit entirely
- [ ] Submit with long axis demonstrated
- [ ] Ensure accurate count possible

### Non-Regional Lymph Nodes

- Distinguish regional vs. non-regional by tumor location
- Non-regional node metastasis = **M1 (distant metastasis)**
- External iliac or common iliac nodes = M1

### Tumor Deposits (pN1c)

**Definition:** Tumor nodule in pericolic/perirectal fat WITHOUT residual lymph node tissue

**May represent:**
- Discontinuous spread
- Lymphovascular spread with extravascular extension
- Completely replaced lymph node
- Perineural invasion (rare)

**Clinical significance:** Reduced disease-free and overall survival

**Documentation:**
- [ ] Relationship to circumferential margin maintained
- [ ] Include CRM if close (<2 mm)

---

## Distant Metastasis (pM)

| Category | Definition |
|----------|------------|
| **M0** | No distant metastasis |
| **M1a** | Metastasis to single organ/site |
| **M1b** | Metastases to >1 organ/site or peritoneum |

### M0 Note

- **Never assign pM0** - this implies pathologic documentation of absence throughout body
- pM0 can only be assigned at autopsy (aM0)
- M0 is a clinical designation

### Common Metastatic Sites

1. Liver (most common)
2. Lung
3. Other colon segments
4. Small intestine
5. Peritoneum

### NOT Distant Metastasis

- Direct extension through serosa into adjacent small intestine, colon/rectum, or adherent organ = **T4b**

---

## Prognostic Factors

Though not required for staging, clinically significant:

- CEA levels (preoperative/pretreatment)
- Tumor deposits (TD)
- Circumferential resection margin (CRM)
- Perineural invasion (PN)
- Microsatellite instability (MSI)
- Tumor regression grade (with neoadjuvant therapy)
- KRAS gene analysis

---

## Terminology Clarification

**Circumferential / adventitial / radial / deep / mesenteric margins** = All refer to non-peritonealized external margin of resection

## QA Checklist

### Pre-Analytic
- [ ] Fixative documented (formalin)
- [ ] Fixation duration documented
- [ ] Cold ischemic time documented (if possible)
- [ ] Specimen integrity assessed

### Macroscopic - Polypectomy
- [ ] Three dimensions
- [ ] Sessile vs. pedunculated
- [ ] Base inked
- [ ] Sectioning documented

### Macroscopic - Resection
- [ ] All component lengths documented
- [ ] Mesentery measured
- [ ] Tumor dimensions (3D)
- [ ] Tumor site/subsite
- [ ] Circumference involvement
- [ ] All margins documented with distances
- [ ] Circumferential margin status
- [ ] Mesorectal completeness (rectal specimens)

### Tumor Assessment
- [ ] Deepest invasion documented
- [ ] Serosal involvement assessed
- [ ] Perforation documented (if present)
- [ ] Relationship to peritoneal reflection (rectal tumors)

### Lymph Node Assessment
- [ ] Total count documented
- [ ] Positive count documented
- [ ] Size range documented
- [ ] Extranodal extension assessed
- [ ] Tumor deposits identified
- [ ] Regional vs. non-regional distinguished
- [ ] Relationship to CRM documented

### Additional Findings
- [ ] Uninvolved colon sampled
- [ ] Polyps/adenomas documented
- [ ] IBD features documented (if present)
- [ ] Treatment effect documented (if applicable)
