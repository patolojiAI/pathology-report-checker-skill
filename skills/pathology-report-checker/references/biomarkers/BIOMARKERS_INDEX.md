# Biomarker Reporting Guidelines

This folder contains reference files for biomarker reporting compliance.

## File Naming Convention

Each biomarker file follows the pattern: `{biomarker}_reporting.md`

Examples:
- `her2_reporting.md`
- `mmr_msi_reporting.md`
- `pdl1_reporting.md`

## Reference File Structure

Each biomarker reference file should include:

1. **Applicable Tumor Types** - Where this biomarker is required/recommended
2. **Testing Indications** - When to test
3. **Required Reporting Elements** - What must be in the report
4. **Scoring Systems** - IHC scores, FISH interpretation, etc.
5. **Quality Metrics** - Controls, adequacy, turnaround time
6. **Cross-Validation Rules** - Internal consistency checks
7. **Turkish Terminology** - Translations
