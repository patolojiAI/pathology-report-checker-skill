# Sinoptik Rapor Şablonları (Türkçe)

CAP tarzı sinoptik rapor şablonları - Türkçe versiyon.

## Şablon Format Standartları

### Alan İşaretleri

| İşaret | Anlam |
|--------|-------|
| `___` | Serbest metin alanı |
| `[ ]` | Onay kutusu (birden fazla seçilebilir) |
| `(_)` | Radyo butonu (sadece bir seçim) |
| `#` | Sayısal değer |
| `*` | Zorunlu/Temel element |
| `+` | Koşullu element |
| `°` | Opsiyonel/Önerilen element |
